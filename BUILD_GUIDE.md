# 빌드 및 실행 가이드 (BUILD GUIDE)

## 1. 개요
* 본 프로젝트(WebMemo)는 서버나 Node.js 환경 등의 별도 설치가 전혀 필요 없는 순수 정적(Static) 웹 애플리케이션이다.

## 2. 실행 방법
1. 프로젝트 루트에 있는 `index.html` 파일을 더블클릭하여 웹 브라우저(Chrome, Edge, Firefox 등)로 실행한다.
2. 만약 외부 라이브러리(Markdown 파서 등)의 CORS 이슈가 발생할 경우, VSCode의 `Live Server` 확장을 사용하거나 파이썬 로컬 서버(`python -m http.server`)를 통해 구동한다.
